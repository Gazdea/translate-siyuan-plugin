# LibreTranslate для SiYuan

Плагин SiYuan для интеграции LibreTranslate для перевода текста непосредственно в редакторе.

## Возможности

- **Несколько способов перевода:**
  - Контекстное меню на выделенном тексте
  - Горячая клавиша (Ctrl+Shift+T по умолчанию)
  - Кнопка в TopBar
  - Команда слеша (/translate)

- **Настройки:**
  - Настройка URL сервера LibreTranslate
  - Опциональный API ключ
  - Языки по умолчанию (исходный и целевой)
  - Настраиваемая горячая клавиша

- **Диалог перевода:**
  - Исходный и переведённый текст рядом
  - Автоопределение языка
  - Обмен языками
  - Замена оригинального текста
  - Копирование в буфер обмена

## Требования

- Приложение для заметок SiYuan (версия 3.5.10+)
- Сервер LibreTranslate (локальный или удалённый)

## Установка

### Из исходного кода

1. Клонируйте репозиторий в папку плагинов SiYuan:
   ```
   {workspace}/data/plugins/translate-siyuan-plugin
   ```

2. Установите зависимости:
   ```bash
   pnpm install
   ```

3. Соберите плагин:
   ```bash
   pnpm run build
   ```

4. Включите плагин в SiYuan: Настройки → Плагины → Загруженные

## Настройка

1. Откройте настройки плагина через кнопку в TopBar или меню плагина
2. Настройте URL вашего сервера LibreTranslate (по умолчанию: `http://localhost:5000`)
3. При необходимости добавьте API ключ
4. Установите языки по умолчанию
5. Проверьте соединение

## Использование

### Через контекстное меню
Выделите текст в редакторе → Нажмите правую кнопку мыши → Нажмите "Перевести"

### Через горячую клавишу
Выделите текст → Нажмите Ctrl+Shift+T (или настраиваемую горячую клавишу)

### Через TopBar
Нажмите на иконку перевода в TopBar для открытия диалога перевода

### Через команду слеша
Введите `/translate` в редакторе для открытия диалога перевода

## Разработка

```bash
# Установить зависимости
pnpm install

# Режим разработки
pnpm run dev

# Сборка для продакшена
pnpm run build

# Линтинг
pnpm run lint
```

## Ссылки

Этот плагин использует [LibreTranslate API](https://libretranslate.com/):
- `POST /translate` - Перевести текст
- `GET /languages` - Получить поддерживаемые языки
- `GET /detect` - Определить язык
- `GET /health` - Проверка работоспособности
